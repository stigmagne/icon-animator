# Contributing to Icon Animator

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork and clone the repo
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the dev server
4. Make your changes in a new branch

## Guidelines

- **Keep it simple** — the editor uses only React and Tailwind, no extra dependencies
- **Test your changes** — make sure `npm run build` passes before submitting
- **One thing at a time** — keep PRs focused on a single feature or fix

## Submitting Changes

1. Create a pull request with a clear description
2. Make sure the build passes
3. Wait for review

## Reporting Bugs

Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser and OS info
